# Groupe 2 — Clients

## Objectif:
Implémenter la gestion des clients (ajout, liste, recherche) avec tableaux dynamiques et pointeurs,
sans utiliser de structures avancées (map, smart pointers, fichiers, etc.).

## Fichiers données:
- include/Client.h
- src/Client.cpp
- tests/test_clients.cpp
- docs/g2-clients.md

## Fonctionnalités:
- Ajouter un client
  - id auto-incrémenté
  - champs : nom, prénom, téléphone
- Lister les clients
- Rechercher un client par id
